Nadec-LinQ2FA

Nadec provides One Time Password (OTP) mobile phone-based two-factor authentication method to secure access to sensitive data and applications. By leveraging the user's existing phone, Nadec-LinQ2FA offers best convenience for users to protect NADEC systems by using their Smart Phone at any time anywhere.

Two-Factor Authentication solution is used to verify a person's identity for security purposes.
